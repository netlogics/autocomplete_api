# Autocompleter

## Install & Run
1. `git clone https://github.com/netlogics/autocomplete_api.git`
2. `cd autocomplete_api`
3. `bundle install`
4. `shotgun config.ru`
5. [browse to localhost:9393/api/v1/autocomplete/search/search-term](http://localhost:9393/api/v1/autocomplete/search/food)


## Run Tests
`rspec`
